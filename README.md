# CS320
### How can I ensure that my code, program, or software is functional and secure?
I ensured that my software is functional by implementing thorough unit testing using JUnit 5. Each class and service is tested for valid and invalid inputs, boundary conditions, and expected behavior. Security and reliability were supported through strong input validation, null checks, and enforcing field length constraints. 

### How do I interpret user needs and incorporate them into a program?
I interpreted user needs by carefully reviewing the provided requirements and translating them into specific functional behaviors within the code. For example, the project required specific variables to be non-null. This requirement was directly translated into code and verified through unit tests. 

### How do I approach designing software?
My approach to designing software begins by understanding the requirements and identifying the major components needed for the system. I then design the classes and their responsibilities. After implementation, I validate the design through unit testing to ensure each component behaves correctly both individually and as part of the overall system.
